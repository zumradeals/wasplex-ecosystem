# Langage et contenu d'interface

**Statut :** spécification proposée — DS-0001

## Voix

Wasplex parle comme un partenaire honnête : phrases courtes, verbes concrets, chiffres visibles et aucune fausse urgence.

## Structure d'un message d'état

1. ce qui s'est passé ;
2. ce qui ne s'est pas encore passé ;
3. état de la valeur ou donnée ;
4. prochaine action ;
5. délai ou source lorsque connu.

Exemple :

> Ton retrait a été transmis à l'opérateur. Le paiement n'est pas encore confirmé. Les 5 000 WP restent réservés. Tu n'as rien à recommencer.

## Erreurs

Éviter :

- « Une erreur est survenue » sans détail utile ;
- « Échec » si le résultat est inconnu ;
- blâme de l'utilisateur ;
- jargon technique ;
- code seul.

Préférer :

> Nous n'avons pas encore reçu la confirmation de l'opérateur. Tes WP restent réservés pendant la vérification.

## Conditions

Une récompense potentielle s'écrit :

> Jusqu'à 30 WP si tu regardes cette publicité jusqu'à la fin et si la vue est validée.

Elle ne s'écrit pas :

> Regarde et gagne 30 F garantis !

## Nombres

Toujours préciser unité :

- 275 WP ;
- ≈ 275 FCFA ;
- frais : 50 FCFA ;
- montant reçu : 4 950 FCFA.

Les grands nombres utilisent espaces insécables selon la langue.

## Traduction

Les clés de traduction suivent l'intention, pas l'emplacement visuel. Les variables ont un type et un exemple. Les textes juridiques et critiques utilisent une revue dédiée.
