# Wasplex UI Catalogue

Catalogue exécutable défini par FND-00-01. Front-end React/TypeScript autonome, sans backend ni donnée réelle.

## Commandes

```bash
npm install
npm run dev
npm run build
npm test
```

## Frontière

- Les contrats normatifs sont dans `../ux/lots/L00/`.
- Les données de ce catalogue sont synthétiques.
- Aucun service de production, Supabase ou Lovable Cloud ne doit être connecté.
- Les composants générés par une IA doivent être revus et normalisés avant intégration.
