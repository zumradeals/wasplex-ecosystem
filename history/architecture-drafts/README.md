# Brouillons d'architecture archivés

Ces fichiers sont des brouillons historiques **non normatifs**. Ils ont été retirés du dossier `architecture/` parce qu'ils réutilisaient des identifiants désormais attribués à des ADR adoptés dans `decisions/`.

Ils sont conservés pour la traçabilité. Ils ne doivent servir ni de base d'implémentation ni de source d'autorité. Pour toute décision active, consulter `decisions/ADR-xxxx-*.md`, puis les spécifications d'application dans `architecture/`.
