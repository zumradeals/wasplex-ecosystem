# Parcours utilisateur

**Statut :** spécification proposée — UX-0001

## Parcours prioritaires P0

- SOS
- récupération de compte compromis
- retrait et résultat inconnu
- alerte nationale critique
- accès aux numéros officiels

## Parcours prioritaires P1

- inscription
- consentements
- publicité rémunérée
- abonnement
- Fonds Social
- Carte Wasplex
- déclaration et restitution
- contestation

## Fiche type

Chaque parcours détaillé ultérieur utilise :

- ID ;
- objectif humain ;
- acteur ;
- préconditions ;
- déclencheur ;
- chemin principal ;
- variantes ;
- états ;
- données ;
- autorisations ;
- écritures ;
- notifications ;
- hors ligne ;
- accessibilité ;
- recours ;
- preuve de réussite.

## Continuité

Un parcours traversant authentification, paiement ou KYC conserve une intention signée et limitée. Au retour, les prix, droits et états sont revalidés avant confirmation.

## Transparence

Le parcours affiche la source de toute valeur :

- campagne ;
- promotion Wasplex ;
- contribution ;
- pool Carte ;
- remboursement ;
- correction.

Aucun solde n'apparaît sans origine consultable.
